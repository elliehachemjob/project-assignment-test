# Article News Finder

Article News Finder is a nice application to use for the user to login as well as searching for his favorite articles.

## How to run the app

1. Either fork or download the app and open the folder in the cli
2. Install all dependencies using the `npm i` command
3. Start the web server using the `npm run` command. The app will be served at http://localhost:3000/
4. Go to http://localhost:19006/ in your browser and start the web version of expo at http://localhost:19006/.

## How to use the library

1. Simply Login.
2. Search for the article you want.

## Features

- Dashboard Component

  - Just give your creditetials and login.
